## Peek-a-Boo

**Description:** Peek-a-Boo Brings 3D Touch to Non-3D-Touch Devices. Peek-a-Boo should be compatible with iOS 9.0 - 11.0. If you have a bug or issue please file it under the 'Issues' tab of the repository.
